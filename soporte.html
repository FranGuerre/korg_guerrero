Preguntas frecuentes

¡Utiliza nuestros filtros!

    • Parece que solo tengo 50 de los 80 ajustes preestablecidos de fábrica en mi Monologue.

      1. Comience con el Monologue apagado.
      2. Mientras mantiene presionado el botón WRITE y el botón EXIT, encienda el monólogo. Se indicará y resaltará “FACTORY RESET” en la pantalla. NO gire la perilla PROGRAM / VALUE ya que esto seleccionará otra opción.
      3. Presione el botón WRITE.
      La pantalla indicará el mensaje "¿Está seguro?".
      4. Gire la perilla PROGRAM / VALUE para seleccionar “Yes” y presione el botón WRITE.
      SUGERENCIA Presione el botón EXIT, o seleccione “No” usando la perilla PROGRAM / VALUE y presione el botón WRITE para cancelar la operación. Nunca apague el Monologue mientras se cargan los datos de fábrica.

    • ¿Cómo puedo guardar mi programa en un destino diferente al actual en mi Volca FM?

      Después de presionar GUARDAR, puede elegir el destino girando la perilla PROGRAM. Después de elegir su destino, presione GUARDAR nuevamente para ejecutar el proceso de guardado o presione EDITAR para salir sin guardar.
      * Esta función se agregó a partir de la versión 1.02 en adelante. volca fm: Actualizador del sistema

    • ¿Cómo restaurar ajustes de fábrica en el Korg Krome?

      Antes de cargar los datos de pre-carga ir a "modo Global P0: Basic Setup-system" y desactivar el seteo “Memory protect” para los datos que queremos cargar. Ej: Programas.
      Cargando los datos de pre-carga se reescribirán los contenidos de la memoria interna. Si queremos mantener los datos existentes en la memoria interna debemos usar “Save all (PCG&SNG)” o “Save PCG” para guardar nuestros datos en una tarjeta de memoria antes de continuar el proceso.

      1) Acceder al modo Global - P0 BASIC SETUP-BASIC PAGE. (Entrar al modo Global presionando el botón Global)
      2)Presionar el botón de menú y elegir “LOAD PRELOAD/DEMO DATA”
      3) En el campo ”Kind” seleccionar All (Preload PCG and Demo songs)
      4) Presionar el botón Ok para cargar los datos pre-cargados si decidimos no hacerlo presionar el botón Cancel. Cuando presionamos el botón Ok, un cuadro de dialogo nos pedirá la confirmación para ejecutar la carga.

    • ¿Qué tipo de datos se guarda una canción grabada del i3?

      La canción se guarda como datos WAV con el nombre AudioSong_xxx en la carpeta AudioRec dentro de la carpeta i3 de la unidad flash USB. Un minuto de grabación será aproximadamente 10 MB de datos.

    • ¿Cuáles son las diferencias entre el Electribe Sampler y el Electribe?

      El muestreador electribe y electribe tienen esencialmente las mismas funciones, pero sus motores de sonido son significativamente diferentes. Electribe: Sintetizador - Contiene un motor de sonido de modelado analógico con 55 sonidos diferentes (el muestreador electribe solo tiene 16)
      - Contiene 16 tipos diferentes de filtro (el muestreador electribe solo tiene tres)
      Electribe Sampler: Sampler
      - Puede usar cualquier sonido por muestreo (el electribe usa solo muestras preestablecidas)
      - Admite muestras estéreo y corte de muestras (el electribe no lo hace)
      Sin embargo, ambos modelos contienen un motor de sonido de modelado simple y muestras de batería estándar, por lo que es fácil hacer ritmos con cualquiera de los modelos. Elija el modelo que mejor se adapte a los sonidos que más utiliza.

    • No puedo tocar seis notas aunque KEY ASSIGN MODE esté configurado en POLY en mi iPolysix

      Puede utilizar la perilla VOICES para especificar la polifonía máxima. Asegúrese de que esta perilla esté en 6.

    • ¿Puedo tocar el Odyssei con un teclado MIDI externo?

      ¡Por supuesto! Puede reproducir el módulo utilizando un teclado MIDI externo como microKEY 2 o nanoKEY2, taktile, etc. a través del kit de conexión de cámara para iPad / adaptador de cámara USB Lightning. Y soporte de conexión micro KEY Air o nanoKEY Studio a través de Bluetooth MIDI.

    • ¿Puedo hacer zoom en el panel de vista?

      En el modo SYNTH, puede tocar dos veces la sección AMPLIFICADOR CONTROLADO POR VOLTAJE del MS-20 para encender "ZOOM MS-20" y cambiar a "PANEL DE CONTROL". Al tocar dos veces el área del logotipo en la sección de parches, puede cambiar a "PATCH PANEL". También puede tocar dos veces para volver desde "PANEL DE CONTROL" o "PANEL DE PATCH".
